# Agrinho_2026
1. O primeiro código (index.html)
É o esqueleto e o conteúdo do site. Ele carrega toda a parte estrutural (textos, títulos, tabelas e seções) e ensina detalhadamente os conceitos técnicos da irrigação com fertilizantes, dividindo-se em:

Aulas Teóricas: Explica a fisiologia das raízes e os tipos de nutrientes (NPK).

Engenharia e Operação: Detalha os equipamentos injetores (como o Venturi) e mostra fórmulas práticas de cálculos de dosagem.

Manutenção: Ensina como evitar o entupimento de gotejadores causado por reações químicas ou algas.

Nota: Ele possui um bloco de estilo interno (<style>) que dita o visual básico desta página especificamente.

2. O segundo código (style.css)
É a identidade visual e o design (folha de estilo externa). Ele serve para pintar, organizar e ditar o comportamento estético das páginas. Seus principais pontos são:

Paleta de Cores Temática: Usa variáveis CSS (:root) com tons de verde-floresta, verde-folha e azul-água para criar uma atmosfera totalmente ligada à agrotecnologia.

Componentes Avançados: Define o visual moderno dos cartões flutuantes (cards com efeito hover), tabelas zebradas e blocos de alerta personalizados.

Responsividade: Garante, através de Media Queries (@media), que todo o conteúdo se ajuste perfeitamente e fique legível em telas de celulares, tablets ou computadores.

Em resumo: O primeiro é a informação (o que o site diz) e o segundo é a apresentação (como o site se mostra visualmente para o usuário).
